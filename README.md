# Chatbot de Trading — Projeto Acadêmico

Chatbot desenvolvido com Python que simula operações de compra e venda 
de criptomoedas em uma corretora fictícia. O usuário interage via chat, 
insere moedas e executa ordens de compra de forma conversacional.

## 🛠️ Tecnologias

- **Python** — lógica do chatbot e regras de negócio
- **Docker** — containerização da aplicação
- **API REST** — comunicação entre front-end e back-end
- **HTML/CSS/JS** — interface do chat

## ⚙️ Funcionalidades

- Responde perguntas sobre criptomoedas em tempo real
- Permite inserir e gerenciar moedas na carteira
- Simula ordens de compra em corretora fictícia
- Ambiente containerizado com Docker para fácil execução

## 📚 Contexto

Projeto acadêmico desenvolvido durante a graduação em Análise e 
Desenvolvimento de Sistemas na FATEC Sorocaba.
